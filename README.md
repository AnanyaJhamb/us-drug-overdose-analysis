# U.S. Drug Overdose Crisis Analysis 

## Project Overview
Comprehensive data analytics project examining the U.S. drug overdose epidemic using multivariate regression analysis. Analyzed 10 years of data covering 91K+ deaths to identify key factors contributing to the 31% year-over-year surge and provide actionable healthcare policy recommendations.

## Key Results

| Metric | Finding |
|--------|---------|
| Time Period | **10 Years** |
| Deaths Analyzed | **91,000+** |
| YoY Surge | **31%** |
| Model R² | **60%** |
| Depression Correlation | 1% increase = **1.46 additional deaths per 100K** |

## Tech Stack
- **Languages:** Python, R
- **Libraries:** Pandas, NumPy, Statsmodels, Scikit-learn, Matplotlib, Seaborn
- **Techniques:** Multivariate Regression, Statistical Analysis, Data Visualization
